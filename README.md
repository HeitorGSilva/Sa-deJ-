# SaúdeJá
## Descrição
SaúdeJá é um protótipo pensado para aquelas pessoas que por falta de um transporte, problemas de saúde ou que moram muito longe de uma unidade de saúde terem como ter acesso a disponibilidade de exames/medicamentos nas unidades da cidade ou região assim evitando que a pessoa se desloque a unidade em vão
